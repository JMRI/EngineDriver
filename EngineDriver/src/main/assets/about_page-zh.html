这款免费的安卓应用程序连接到WiThrottle 服务器。支持的服务器包括 JMRI， DCC-EX EX-CommandStation（使用 WiThrottle 协议或原生 DCC-EX 命令协议）， MRC Prodigy WiFi， Digitrax LnWi， WifiTrax（适用于 NCE） 和 IoTT Stick 。连接后，您可以控制您的机车 以及您的整个模型铁路。支持速度、方向和多达 32 项 DCC 功能，适用于 1 至 6 台机车或编组。您可以创建和编辑编组（软件定义）。您还可以 控制布局电源、道岔、路线，并访问 JMRI Web 面板和窗口。

先决条件 (适用于JMRI):

验证您的Android设备能否通过WiFi连接到您的网络并获取IP地址。
确保您的个人电脑上运行的是最新版本的JMRI（2.8版本可以，3.0及以上版本效果更好），并且已连接到您的布局。启动JMRI，并验证您能否使用附带的“Throttle”应用程序来控制布局中的机车。

启动JMRI的WiThrottle功能，该功能位于DecoderPro的“工具”选项卡下的“节流阀”部分。同时，强烈建议您启动JMRI Web Server功能（在“工具”选项卡下点击“启动JMRI Web Server”）。

先决条件 (适用于MRC Wifi):

连接 MRC WiFi 模块，并验证您的设备是否可以连接到其网络并获取 IP 地址。

先决条件（适用于 Digitrax LnWi）：

连接 LnWi，并验证您的设备是否可以连接到其网络并获取 IP 地址。

先决条件（适用于 DCC-EX EX-CommandStation）：

Engine Driver 可以直接连接到处于 Station 模式或 Access Point 模式的 DCC-EX EX-CommandStation， 连接方式可以是使用 wiThrottle 协议，也可以是使用 DCC-EX 原生命令协议。 Engine Driver 默认使用 wiThrottle 协议。如果您希望使用 DCC-EX 原生命令， 则必须更改 Engine Driver 的设置。
如果您使用 DCC-EX 原生命令，则会有一个额外的菜单项 (DCC-EX)，用于 编程 CV 和其他功能。
注意：如果您通过 JMRI 连接 EX-CommandStation，则必须使用 wiThrottle 协议，因此 其他功能将不可用。

配置：

在手机上启动 Engine Driver 应用，等待服务器地址出现在“已发现的服务”中。

点击已发现的地址，或者您可以直接输入 WiThrottle 服务器的地址和端口（使用 WiThrottle 窗口中显示的值）。
您可以根据需要启用“自动连接到第一个 WiThrottle 服务器？”首选项。此外，还有一个位于 jmri.mstevetodd.com、端口为 44444 的“演示”服务器，您可以连接到该服务器进行测试。

在油门屏幕上，点击每台机车对应的“选择”按钮。
在下一个屏幕上，选择选择方式（DCC地址输入、JMRI名录或最近使用的机车）。选择机车后，您将返回油门屏幕。
重复上述步骤，点击其他“选择”按钮以选择更多列车。
注意：按住名录中的某个项目可查看更多详细信息。

使用滑块（或音量控制）（或可选的箭头按钮）控制火车的速度，使用后退、停止和前进按钮控制方向。
要释放所有火车，只需按下“返回”按钮并退出应用程序。这也会停止火车（可在偏好设置中更改）。您可以通过点击速度指示器（小“v”图标表示哪个方向控制音量）来重新分配音量控制。

要“即时”创建编组，只需为同一油门选择其他机车即可。系统会提示您输入每台新机车相对于现有编组的行驶方向。所有机车的行驶方向和速度都会被发送。按下“释放”按钮将释放该油门控制的所有机车。

按下手机的菜单按钮，即可进入道岔、路线和功率控制屏幕，以及调整设置和设置各种偏好。

您还可以向左或向右“甩动”，在油门、路线和道岔页面之间快速来回跳转。

ESU MobileControl Pro 操作：

Engine Driver 应用程序完全支持 ESU MobileControl Pro 安卓油门。

除了可以通过控制旋钮控制所选油门的速度外，还可以通过零位限位开关切换方向。

最多可同时控制三列火车——当前由控制旋钮操控的火车会在速度指示器上显示一个小的“v”字样。点击相应的速度指示器即可切换正在操控的火车。也可以将设备侧面的某个按钮设置为切换到下一列正在操控的火车——默认情况下，右下角的按钮已分配给此功能。

当被分配控制列车时，绿色 LED 指示灯亮起；当没有分配列车时，绿色 LED 指示灯闪烁。

此外，您可以使用“停止”按钮暂停当前运行的列车，或停止所有受控列车。短按“停止”按钮（默认小于0.5秒）可暂停当前受控列车的运行。红色LED指示灯将快速闪烁以示暂停。再次按下“停止”按钮，当前运行的列车将恢复到之前设定的速度。长按“停止”按钮（默认大于0.5秒）可停止所有受控列车。红色LED指示灯将缓慢闪烁以示停止。再次按下“停止”按钮，所有受控列车将保持静止状态，此时您可以再次控制当前选定的列车。

在停止模式下，屏幕上的速度更改按钮和进度条将对当前停止的油门禁用。控制油门之间的切换功能也将被禁用。

可以对设备侧边按钮进行配置，使其执行各种操作。默认设置如下：

  • 左上角 - 提高速度
  • 左下角 - 降低速度
  • 右上角 - 切换方向
  • 右下角 - 选择下一个节流阀

这些设置可以通过“首选项”中的“ESU MobileControl Pro 选项”页面进行修改。

配置：

在“首选项”中，您可以找到一系列丰富的选项，自定义 EngineDriver 的屏幕和操作，使其与您的设备、布局和个人风格完美匹配。这些选项包括油门布局、快速时钟显示、速度步进选项、网页视图、按钮大小、附加按钮、滑块大小和行为、要显示的油门数量、松开油门时停止、允许/禁止实时连接、自动连接等等。

如果您已在服务器中为花名册条目输入了功能标签，则将使用这些标签。否则，您可以使用“功能默认值”菜单选项来设置要显示的默认标签和功能编号。要移除按钮，请将其标签涂黑。

在 JMRI 计算机端，请检查您的 WiThrottle 设置，确保您已启用或禁用所需的控制功能。如果您已在面板中定义了道岔和线路，请务必为希望在油门上显示的道岔和线路指定“用户名”。您还可以使用 WiThrottle 窗口中的“WiThrottle > 筛选控件”选项来筛选显示的道岔。建议启用“使用紧急停止”功能。

在 JMRI 中输入列车员名册、定义线路和道岔，都能让您的 Engine Driver 油门控制更加强大易用（虽然并非必需）。如果 JMRI Web 服务器已启动，EngineDriver 将显示更多信息（例如列车员名册详情和图标图像）。
